Based on the book "Ruby on Rails - Up and Running" (by Bruce A. Tate & Curt Hibbs), this tool automates the process of following the book so that you don't have to type any commands or edit any files, but also modifies all examples to work with Rails 3. All you need to do is specify the page number where you're reading and the tool will issue all commands and edit all files to bring the state of your system to an equivalent state at that page.
